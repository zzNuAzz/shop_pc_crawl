# shop_pc_crawl


## Start


chạy 

	npm install 
	
để install các dependencies có trong file package.jon 

Tiếp theo chạy 

	npm run [tên trang cần lấy dữ liệu]
  
 có thể là "anphat", "hanoicomputer", "gearvn"
 
 danh sách category cần craw miêu tả ở trong file category.json.
 kết quả lưu ở /data
